# 一、 通讯方式

清华索机器人计算机作为通讯服务器，安装于地面站；中建四局计算机作为通讯客户 端，安装于索机器人动平台上。 

# 二、通讯方式

通讯采用类似于 Modbus TCP 问答式，即客户端向服务器先发送通讯数据包，服务器 接收到通讯包后，向客户端发送返回数据包。 

通讯端口号：2001。
