# Agent Memory Entry Template

复制本模板到 `session_log.md` 顶部，并替换尖括号内容。

```markdown
## YYYY-MM-DD HH:MM - <标题>

<!-- AGENT-MEMORY: entry -->

### 摘要

- <这次会话最重要的先验知识。>

### 影响范围

- `<关键文件或目录>`

### 关键决策

- <为什么这样做，以及后续 agent 不应误改的边界。>

### 验证证据

- `<运行过的命令>`：<结果。>

### 后续注意

- <仍需复测、现场确认或禁止恢复的内容。>
```
