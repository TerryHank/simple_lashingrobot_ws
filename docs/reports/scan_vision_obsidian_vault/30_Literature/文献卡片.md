# 文献卡片

## Rebar-tying Robot based on machine vision and coverage path planning

- 年份：2024
- 来源：Robotics and Autonomous Systems, Vol. 182, Article 104826
- DOI：<https://doi.org/10.1016/j.robot.2024.104826>
- 页面：<https://pure.bit.edu.cn/en/publications/rebar-tying-robot-based-on-machine-vision-and-coverage-path-plann>

要点：

- 面向钢筋绑扎机器人。
- 使用 RGB-D 图像。
- 提出 Hough transform multi-segment fitting 检测钢筋交点。
- 报告交点检测准确率 99.4%，检测误差 2.8 mm 以内。

对本项目的启发：

- 扫描层不应只做规则网格生成，应引入真实线段/交点检测证据。
- Hough 多段拟合可以作为现有 S2 的局部验证或替代方案。

## YOLO-FAS

- 年份：2025
- 来源：Neurocomputing, Vol. 624, Article 129485
- DOI：<https://doi.org/10.1016/j.neucom.2025.129485>
- 页面：<https://www.sciencedirect.com/science/article/pii/S0925231225001572>

要点：

- 面向绑扎机器人上的钢筋交点与绑扎状态检测。
- 基于 YOLOv5s，并做轻量化、剪枝、量化和 TensorRT 加速。
- 真实施工场景中交点识别准确率超过 98%。

对本项目的启发：

- 如果传统几何算法在现场噪声下不稳定，可引入轻量关键点检测模型作为候选验证器。
- 训练数据和增强很关键，尤其要覆盖暗光、反光、遮挡和不同钢筋直径。

## Vision-Based Detection, Localization, and Optimized Path Planning for Rebar Intersections

- 年份：2025
- 来源：Applied Sciences, 15(13), 7186
- DOI：<https://doi.org/10.3390/app15137186>
- 页面：<https://www.mdpi.com/2076-3417/15/13/7186>

要点：

- 结合 RGB-D 传感、相机标定和坐标转换。
- 使用深度范围过滤、Gaussian smoothing、Hough Transform 检测交点。
- 关注从像素坐标到机器人坐标的转换。

对本项目的启发：

- 坐标精度不只取决于 2D 识别，还取决于深度取点、外参和局部 3D 稳定性。
- 当前项目应把局部 ROI 内的 3D 拟合纳入质量门。

## A Novel Grid Intersection Point Detection And Matching

- 年份：2016
- DOI：<https://doi.org/10.21307/ijssis-2017-869>
- 页面：<https://sciendo.com/article/10.21307/ijssis-2017-869>

要点：

- 针对网格图像交点检测。
- 先用 Canny 与 Hough 提取网格边界线段。
- 通过聚类获得全局结构参数。
- 再按全局结构划分局部检测区域，在局部区域内用 ridge line fitting 精定位交点。

对本项目的启发：

- 适合改造当前 PR-FPRG：全局周期只负责提供候选区域，最终点必须由局部 ridge 拟合确认。

## LSD: a Line Segment Detector

- 年份：2012
- DOI：<https://doi.org/10.5201/ipol.2012.gjmr-lsd>
- 页面：<https://www.ipol.im/pub/art/2012/gjmr-lsd/>

要点：

- 线性时间线段检测。
- 亚像素结果。
- 使用 a contrario 验证控制误检。

对本项目的启发：

- 可用作 rectified response 或 IR 图上的线段候选提取器。
- 比单纯 profile 铺线更适合做「真实线段支撑」。

## Steger: An unbiased detector of curvilinear structures

- 年份：1998
- DOI：<https://doi.org/10.1109/34.659930>
- 页面：<https://cir.nii.ac.jp/crid/1361699995960273920>

要点：

- 经典曲线/脊线无偏检测方法。
- 适用于细长结构中心线检测。

对本项目的启发：

- 钢筋在深度响应或 IR 响应中可以看作脊线结构。
- 可用于局部交点周围的线中心精定位。

## Frangi: Multiscale vessel enhancement filtering

- 年份：1998
- DOI：<https://doi.org/10.1007/BFb0056195>
- 页面：<https://cir.nii.ac.jp/crid/1363388845956994688>

要点：

- 用 Hessian 特征值做多尺度管状结构增强。
- 常用于血管等细长结构检测。

对本项目的启发：

- 可把钢筋当作多尺度线状结构增强，再做 skeleton / Hough / 交点检测。
- 适合处理钢筋直径变化、局部宽窄变化。

## Advanced Vision-guided Robotic Tying with 6-DoF Rebar Pose Estimation

- 年份：2025
- 来源：ISARC 2025
- 页面：<https://www.iaarc.org/publications/fulltext/028_Advanced_Vision-guided_Robotic_Tying_with_6-DoF_Rebar_Pose_Estimation.pdf>

要点：

- 使用深度学习关键点检测与点云配准。
- 关注复杂空间姿态下的绑扎点和 6-DoF 姿态估计。
- 提到基于像素关键点和深度值计算三维坐标，再做 coarse-to-fine registration。

对本项目的启发：

- 当前项目如果要从平面绑扎走向更复杂场景，最终需要从「点」扩展到「局部钢筋姿态」。
- 局部点云配准可以作为执行层精修方向。
